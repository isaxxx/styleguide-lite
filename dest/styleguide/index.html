Requires

  • RubyGems 2.4.4 or later

  • Bundler 1.7.9 or later

  • Node.js 0.10.35 or later

  • NPM 2.2.0 or later

  • Grunt-CLI 0.1.13 or later

  • Bower 1.3.12 or later

Getting Started

  1. Download

Github

  1. Change into a new directory
$ cd whisk-x.x.x
  1. Install Node Package Modules
$ npm install
  1. Setup Frontend Libraries
$ grunt bower-concat
  1. Starting
$ grunt

Specification

Grunt Tasks

default

To Watch sass and coffee directories, compile the files.

bower-concat

Collected the libraries in Bower, further join them.

minify

Compresses files, and generate the GZIP.

generate-styleguide

Generate styleguide by kss-node.

generate-yuidoc

Generate yuidoc.

generate-sprite

Generate the spritesheet and their CSS files by grunt-spritesmith.

generate-embed

Generate the data URI scheme of images in embed directory by grunt-data-uri.

check-bless

Checking the number of CSS selectors by Bless.

check-csscss

Checking the rules set of CSS that have duplicate declaration by csscss.

lint-css

Checking CSS syntax by CSS Lint.

lint-js

Checking Javascript syntax by JSHint.

template-wordpress

Generate a default stylesheet of Wordpress.

template-html

Generate a HTML file for using Whisk.
